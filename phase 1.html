C Programming Fundamentals

C is a powerful general-purpose programming language that is excellent for beginners to learn because it helps you understand how computers work at a fundamental level.

1. Variables and Data Types

Variables are containers for storing data values. In C, each variable must have a specific type, which determines the size and layout of the variable's memory.

Basic Data Types in C:

  • int - integers (whole numbers)
  • float - floating-point numbers (decimals)
  • double - double-precision floating-point numbers
  • char - single characters
#include <stdio.h>

int main() {
    // Variable declaration and initialization
    int age = 25;
    float height = 5.9;
    double salary = 50000.50;
    char grade = 'A';
    
    // Printing variables
    printf("Age: %d\n", age);
    printf("Height: %.2f\n", height);
    printf("Salary: %.2f\n", salary);
    printf("Grade: %c\n", grade);
    
    return 0;
}

2. Operators

Operators are symbols that tell the compiler to perform specific mathematical or logical operations.

Arithmetic Operators

+ (addition), - (subtraction), * (multiplication), / (division), % (modulus)

Relational Operators

== (equal), != (not equal), > (greater than), < (less than), >= (greater or equal), <= (less or equal)

Logical Operators

&& (AND), || (OR), ! (NOT)

3. Control Structures: If-Else and Loops

Control structures allow you to control the flow of execution of your program.

#include <stdio.h>

int main() {
    int number = 10;
    
    // If-else statement
    if (number > 0) {
        printf("The number is positive.\n");
    } else if (number < 0) {
        printf("The number is negative.\n");
    } else {
        printf("The number is zero.\n");
    }
    
    // For loop
    printf("For loop: ");
    for (int i = 0; i < 5; i++) {
        printf("%d ", i);
    }
    printf("\n");
    
    // While loop
    printf("While loop: ");
    int j = 5;
    while (j > 0) {
        printf("%d ", j);
        j--;
    }
    printf("\n");
    
    return 0;
}

4. Functions and Recursion

Functions are blocks of code that perform a specific task. Recursion is a technique where a function calls itself.

#include <stdio.h>

// Function declaration
int add(int a, int b);
int factorial(int n);

int main() {
    int result = add(5, 3);
    printf("5 + 3 = %d\n", result);
    
    // Recursion example
    int fact = factorial(5);
    printf("Factorial of 5 is %d\n", fact);
    
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

// Recursive function to calculate factorial
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

5. Arrays and Strings

Arrays are used to store multiple values in a single variable. Strings are arrays of characters.

#include <stdio.h>
#include <string.h>

int main() {
    // Arrays
    int numbers[5] = {1, 2, 3, 4, 5};
    printf("Third element: %d\n", numbers[2]);
    
    // Strings
    char greeting[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
    char name[] = "Alice";
    
    printf("Greeting: %s\n", greeting);
    printf("Name: %s\n", name);
    printf("Length of name: %lu\n", strlen(name));
    
    return 0;
}

Web Development Basics

Web development involves creating websites and web applications. The core technologies are HTML, CSS, and JavaScript.

1. HTML: Structure, Forms, Tables

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph of text.</p>
    
    <!-- Form example -->
    <form>
        <label for="name">Name:</label>
        <input type="text" id="name" name="name"><br>
        
        <label for="email">Email:</label>
        <input type="email" id="email" name="email"><br>
        
        <input type="submit" value="Submit">
    </form>
    
    <!-- Table example -->
    <table border="1">
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
        <tr>
            <td>Alice</td>
            <td>25</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td>30</td>
        </tr>
    </table>
</body>
</html>

3. JavaScript: Variables, Loops, Functions

JavaScript is a programming language that makes web pages interactive.

// JavaScript example

// Variables (let, const, var)
let message = "Hello, World!";
const pi = 3.14159;
var count = 0;

// Data types
let name = "Alice"; // String
let age = 25; // Number
let isStudent = true; // Boolean
let scores = [95, 89, 72]; // Array
let person = { // Object
    firstName: "John",
    lastName: "Doe",
    age: 30
};

// Functions
function greet(name) {
    return "Hello, " + name + "!";
}

// Arrow function
const multiply = (a, b) => a * b;

// Loops
console.log("For loop:");
for (let i = 0; i < 5; i++) {
    console.log(i);
}

console.log("While loop:");
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

// Event handling
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});
